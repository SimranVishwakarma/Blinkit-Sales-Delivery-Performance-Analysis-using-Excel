# Blinkit-Sales-Delivery-Performance-Analysis-using-Excel
This project analyzes Blinkit sales and delivery data using Microsoft Excel to identify revenue trends, top-selling products, and delivery performance. Excel formulas, Pivot Tables, and dashboards are used to generate business insights and support data-driven decision making.
This project analyzes Blinkit grocery sales, customer orders, and delivery performance using Microsoft Excel.
The objective is to understand sales trends, product demand, delivery efficiency, and customer behavior through structured data analysis and dashboards.

The project demonstrates strong hands-on skills in Excel formulas, Pivot Tables, data cleaning, and dashboard creation, commonly used by data analysts in real-world business scenarios.
